# Description

The following code snippets present how to save data to the shared preferences and get data from the shared preferences.

## Code

### Save Data

```java
String MY_GLOBAL_PREFS = "my_global_prefs";
SharedPreferences.Editor editor = getSharedPreferences(MY_GLOBAL_PREFS, MODE_PRIVATE).edit();
editor.putString(SigninActivity.EMAIL_KEY, email);
editor.apply();
```

### Get Data

```java
SharedPreferences prefs = getSharedPreferences(.MY_GLOBAL_PREFS, MODE_PRIVATE);
String email = prefs.getString(EMAIL_KEY, "");
```
