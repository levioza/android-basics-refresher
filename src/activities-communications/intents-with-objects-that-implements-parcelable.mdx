# Description

In this example, we create an intent to the `DetailActivity.class`. We specify a key to add an item as extra. Please this item should implement the parcelable imterface. Also, we can get the intent in the second activity using the `getIntent().getExtras().getParcelable(ITEM_KEY)`.

## Code

### Package an intent

```java
Intent intent = new Intent(this, DetailActivity.class);
intent.putExtra(ITEM_KEY, item);
this.startActivity(intent);
```

### Get Intent

```java
DataItem item = getIntent().getExtras().getParcelable(ITEM_KEY);
```
